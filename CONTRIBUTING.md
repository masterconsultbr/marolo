# 🤝 Contribuindo

Bem-vindo! Este é um projeto de aprendizado. Qualquer um pode contribuir.

## Como Contribuir

### 1. Fork e Clone
```bash
git clone https://github.com/seu-usuario/futebol-app.git
cd futebol-app
```

### 2. Crie uma Branch
```bash
git checkout -b feature/sua-feature
```

### 3. Faça Alterações
- Siga o padrão de código existente
- Adicione commits claros e descritivos

### 4. Push e Pull Request
```bash
git push origin feature/sua-feature
```

## Padrão de Commits

```
feat: nova funcionalidade
fix: correção de bug
docs: documentação
refactor: refatoração de código
style: formatação
test: testes
```
